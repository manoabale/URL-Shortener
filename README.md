# URL-Shortener🔗 

A fast and simple **URL Shortener** built with **HTML**, **CSS**, and **JavaScript**.  
This tool allows users to shorten long URLs using a free public API and copy the shortened link instantly.

Perfect for beginners learning API usage, DOM manipulation, and interactive UI design.



## 🚀 Features

### 🔹 Core Features
- Shorten any long URL  
- Uses a free and reliable API (shrtco.de)  
- Copy shortened URL to clipboard  
- Clean and responsive UI  

### 🎨 UI Features
- Modern and minimal design  
- Centered layout  
- Highlighted result box  



## 📂 Project Structure



url-shortener/ │ ├── index.html ├── style.css └── app.js




## 💻 Technologies Used

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla)** – API calls + copy function  

No frameworks required.
